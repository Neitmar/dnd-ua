# Assets для DnD UI

## Необхідні зображення

### Фони пергаменту
- `bg_parchment.jpg` — світлий фон пергаменту (основной)
- `bg_parchment_dark.jpg` — темний фон пергаменту (варіант)

### Крила
- `wing_demon.png` — крило демона (ліве крило з дзеркальним відзеркаленням)
- `wing_angel.png` — крило янгола (праве крило)

## Fallback поведінка

Усі компоненти мають вбудовані fallback-и:
- Якщо зображення не завантажується — використовується кодовий градієнт
- Для крил — малюються векторні крила за допомогою CustomPaint
- Для пергаменту — використовується близький до кольору фон

## Як додати зображення

1. Розмістіть PNG/JPG файли у цій директорії
2. Убедитеся, що вони зареєстровані у `pubspec.yaml` під `assets:`
3. Система автоматично почне використовувати текстури замість кодових fallback-ів
