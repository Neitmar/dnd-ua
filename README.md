# ДнД - Компаньон

[![Flutter](https://img.shields.io/badge/Flutter-3.11.5-blue.svg)](https://flutter.dev/)
[![Dart](https://img.shields.io/badge/Dart-3.11.5-blue.svg)](https://dart.dev/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

**ДнД - Компаньон** — мобільний помічник для гравців у Dungeons & Dragons 5-ї редакції. Підтримує українську, російську та англійську мови.

## ✨ Особливості

- **📝 Створення персонажа** — Повний аркуш персонажа з усіма характеристиками
- **⚔️ Бойова система** — Трекер здоров'я, ініціативи, станів та зброї
- **🎒 Інвентар** — Управління предметами, грошима та екіпіровкою
- **🔮 Заклинання** — Бібліотека заклинань з можливістю підготовки
- **🏰 Арморі** — Візуальна екіпіровка персонажа з усіма слотами
- **🎲 Кубики** — Вбудований кидальник кубиків для всіх типів
- **💰 Гаманець** — Управління валютою (МД, СД, ЕД, ЗД, ПД)
- **🌙 Темна/Світла тема** — Зручний інтерфейс для гри в будь-який час
- **💾 Збереження** — Автоматичне збереження всіх даних локально
- **🌐 Мультимовність** — Українська, Російська, Англійська

## 🚀 Встановлення

### Передумови

- Flutter SDK 3.11.5 або вище
- Dart SDK 3.11.5 або вище
- Android Studio або VS Code з розширенням Flutter

### Клонування репозиторію

```bash
git clone https://github.com/Neitmar/dnd-ua.git
cd dnd-ua
```

### Встановлення залежностей

```bash
flutter pub get
```

### Запуск додатка

#### Android / iOS:
```bash
flutter run
```

#### Web:
```bash
flutter run -d chrome
```

## 🛠️ Технології

- **Flutter** — UI фреймворк для кросплатформної розробки
- **Dart** — Мова програмування
- **Provider** — Стейт менеджмент
- **Shared Preferences** — Локальне збереження даних
- **Material Design 3** — Дизайн система

## 📁 Структура проекту

```
lib/
├── main.dart                  # Точка входу
├── constants/
│   ├── app_assets.dart        # Централізований реєстр асетів
│   └── design_tokens.dart     # Дизайн-токени (кольори, шрифти)
├── providers/
│   └── app_state.dart         # Головний провайдер стану
├── screens/                   # Екрани додатка
├── services/
│   └── localization_service.dart  # Локалізація (uk/ru/en)
├── widgets/                   # Спільні віджети
└── data/                      # Статичні дані
```
